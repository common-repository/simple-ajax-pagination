Just another simple ajax pagination plugin. Simple but flexible.

== Description ==

Simple ajax pagination plugin create simple pagination to ajax pagination with powerful jquery and easy to install and setup. It is also work for custom post types and multi language website.

= Docs & Support =

Create ajax pagination without any hard code(PHP and jQuery).

Just follow few steps and create ajax pagination from simple pagination in any pages.

* Go to Settings -> SA Pagination
* Select page
* Add ID or Class with/without "#" or "." in "Container Div" field(that means this ID or Class contain post and pagination only).
* Add ID or Class with/without "#" or "." in "Pagination Div" field(that means this ID or Class lead acnchor tag of pagination).
* Click on update button to save your record and refresh your listing page.


`Note : Both fields "Container Div" and "Pagination Div" contain "ID" or "Class" with/without "#(id - hash)" or ".(class - dot)".`

== Installation ==

1. Upload the entire `simple-ajax-pagination` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'SA Pagination' menu in your WordPress admin panel under Settings tab.
